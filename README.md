# trending-news-aggregator
A web-based application that scrapes Google News and ranks articles based on user clicks.

## 🚀 Features
- Web scraping using BeautifulSoup
- Click-based trending algorithm
- CSV storage system
- Flask web interface
- Real-time updates

## ⚙️ Run Project
```bash
pip install flask requests beautifulsoup4
python "Real time social media feed with trending algorithm.py"
